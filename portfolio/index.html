現在の取り組み

AgenticAI × 常時センシングTWSによる"常時無意識コミュニケーション"のコンセプト立案からデモ開発まで担当。環境音や行動からユーザーの潜在意識下の文脈(状況)を理解し、パーソナライズされた体験を提供する次世代AIシステムの構築に取り組んでいます。

LangGraph Agentic System Hubは、LangGraphとMCP（Model Context Protocol）ツールを活用したマルチエージェントLLMアプリケーションのための拡張性・モジュール性の高いフレームワークです。音楽検索、天気情報、飲み会会場検索、タスク管理など、日常の多様なユースケースに対応しています。

技術スタック・開発スタイル

  • LangGraph/LangChainによるエージェントオーケストレーション
  • FastAPIやGradio、FlaskによるAPI・Web UI開発
  • Docker・GitHub Actionsを活用したCI/CDパイプライン自動化
  • MCP（Model Context Protocol）による外部ツール連携
  • Python, PyTorch, TensorFlow, React, AWS, Git, Docker, CI/CD, Flask など幅広い技術

CI/CD自動化にも注力しており、GitHub Actionsによる自動テスト・Dockerビルド・デプロイ、Pull Request時のMockテスト、Dockerfileによるマルチステージビルドなど、開発効率と品質向上を両立しています。

今後の展望

  • RAGやGoogle Calendar連携、Notion自動要約、YouTube音源分離など多様な拡張
  • エッジAIやLLMの融合による新しい体験の創出
  • より多様なユースケースに対応するAgentic AI Hubの実現

AI技術を活用した革新的なソリューションの開発を通じて、体験した人の心が揺さぶられ、その人の考えや行動に影響を与えるような価値を提供したいと考えており、結果的に人々の生活をより豊かにできたら嬉しいです。
